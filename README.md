# Curso JavaScript 

## Estrutura do Projeto
- `src/`: Exemplos teóricos das aulas.
- `exercicios/`: Meus exercícios práticos (portfólio).
- `__tests__/`: Testes automatizados com Jest.

## Como Executar
```bash
# Rodar um exemplo:
node src/dia2/exemplo-condicionais.js

# Rodar meus exercícios:
node exercicios/dia2/condicionais.js

# Rodar testes:
npm test
